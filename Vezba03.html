Osnovno o CSS-u

CSS predstavlja veoma mocan jezik pomocu kojeg mozete dodati stil nasim vasim Web stranama.

Osnoovna CSS sintaksa

CSS je predstavljen kao skup pravila koja se pisu ba sledeci nacin:

selektor
{
  atribut1; vrednost1;
  atribut2; vrednost2;
}
